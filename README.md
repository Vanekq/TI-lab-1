# Лабораторная работа №1: Шифры Rail Fence и Виженера

## 🚀 Запуск
Скачайте и запустите `TI-lab-1.exe` (установка не требуется)

## 📌 Возможности
- **Rail Fence**: ключ - число
- **Виженер**: ключ - русское слово
- Чтение/сохранение файлов

## 📖 Использование
1. Выберите метод
2. Введите ключ и текст
3. Нажмите "Зашифровать"/"Дешифровать"
